# 📝 TaskFlow - To-Do List App

A simple and elegant To-Do List web application built with Flask and MySQL.

## 🚀 Features

* Add tasks
* Mark tasks as completed
* Delete tasks
* Dark theme UI

## 🛠️ Tech Stack

* Python (Flask)
* MySQL
* HTML, CSS

## 📂 Project Structure

taskflow/
│── app.py
│── static/style.css
│── templates/index.html
│── templates/add.html

## ⚙️ Setup Instructions

1. Clone the repository
   git clone https://github.com/srirampachava/TaskFlow.git

2. Navigate to project
   cd TaskFlow

3. Install dependencies
   pip install flask mysql-connector-python

4. Setup MySQL database
   Run the SQL file to create table

5. Run the app
   python app.py

6. Open browser
   http://127.0.0.1:5000
